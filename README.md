**Frontend Mini Tasks**

This repository contains a set of simple and structured HTML/CSS practice tasks designed to build a strong foundation in frontend development. Each file represents an individual mini-project focusing on basic web elements, layout, and formatting.

**Included Tasks**

Bio Data – Basic personal profile layout

Class Time Table – Timetable created using HTML tables

Feedback Form – Simple form with input fields

Movie Library – List-based movie collection page

Movie Review – Static review page structure

Naukari Page – Job/profile-style web page

Restaurant Menu – Menu layout using lists

Semester Result – Marks/result table design

Unordered List – Task demonstrating list elements

Custom Attribute – Example using custom HTML attributes

LinkedIn Page – Basic LinkedIn-style profile layout

Technologies Used

HTML5

CSS3

**Purpose**

This repository is created to practice essential frontend concepts and improve understanding of HTML structure, forms, tables, and basic styling.
